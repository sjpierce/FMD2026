## English

This repository holds materials for one of my professional presentations
(Pierce, 2026, July 20). Its primary purpose is for me to use version
control tools while developing my slides but its secondary purpose is
distributing the slides to my audience, which is why the repository is
public.

There are two different types of users of this repository: myself and my
audience. I am the only person editing the code in this repository and I
assume the audience just wants copies of my slides, so I am omitting
information about the structure of the repository and how to render the
slides.

Because the main audience is from Peru, I have used Google Gemini to
translate the slides (including speaker notes) and contents of this page
into Peruvian Spanish. I hope the translation is useful and accurate,
but I don’t know Spanish myself.

## Obtaining the Slides (Cómo obtener las diapositivas)

### English

The link below takes you to a GitHub page with a *Download raw file*
button in the upper right corner of the screen. Its icon looks like an
arrow pointing down into a tray. It is just to the left of the pencil
icon. Use that to download the file to your computer, then open it with
a web browser.

- [`Slides.html`](https://github.com/sjpierce/FMD2026/blob/main/Slides.html)
  *Most current English version of the slides*.

### Español

El enlace de abajo les llevará a una página de GitHub con un botón de
Download raw file en la esquina superior derecha de la pantalla. Su
icono parece una flecha que apunta hacia abajo dentro de una bandeja.
Está justo a la izquierda del icono del lápiz. Utilícenlo para descargar
el archivo en su computadora y luego ábranlo con un navegador web.

- [`Diapositivas.html`](https://github.com/sjpierce/FMD2026/blob/main/Diapositivas.html)
  *Versión en español más reciente de las diapositivas*.

## Usage Tips (Consejos de uso)

### English

- After you open the `Slides.html` file on your computer, press the `s`
  key on your keyboard to see the speaker view that contains my speaker
  notes.
- There are also other [slide navigation keyboard
  shortcuts](https://quarto.org/docs/presentations/revealjs/presenting.html).

### Español

- Después de abrir el archivo `Diapositivas.html` en su computadora,
  presionen la tecla `s` en su teclado para ver la vista del orador que
  contiene mis notas de orador.
- También hay otros  
  [atajos de teclado para navegar por las
